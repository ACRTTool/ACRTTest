ACRT USER GUIDE

Create Report page

  1. Select Create Report in the menu to begin a new Report
  2. Select Select template or test results file and load JSON file:
    • First time use SuccessCriteriaTT4.json or SuccessCriteriaTT5.json located in the 'Resources' folder.
      • Or
    • Use previously created JSON report file.
  3. Select Load selected file button.
  4. In 'Create Report' page each field has characters limit, you can not enter more than it's limit.
  5. Open Product and Testing Information to complete each form and the required fields.
  6. It is recommended to fill optional fields as well.
  7. Add/Edit test results
  8. Select Save file by selecting Save button or the keyboard shortcut (Alt+s).

View Report

  1. Select View Report in the top menu to open an existing report that contains at least one valid test result
  2. Select Select template or test results file and load JSON file:
  3. This is read only page, no edit/update allowded here.
  4. Select Save as HTML button or keyboard shortcut (Alt+s)to generate a printer-friendly version and keep JSON file for future edits on the Create Report page.

Instructions for ACRT

  1. After ACRT is downloaded, unzip/extract files and open index.html (preferred browsers are Firefox, Microsoft Edge, or Google Chrome).
  2. To get detail ACRT Training Manual, please visit Instructions
  3. You can also get accessible ACRT Training Manual in both word and pdf file inside 'Resources' folder.
  4. Select 'Create Report' to create or edit test result.
  5. Select and load either standard template JSON file (TT4 or TT5 JSON file from 'Resources' folder for the first time) or valid custom JSON file (one you created for editing).
  6. Once filled/edited all required fields, you will see option to save test result.
  7. To save file changes, select the keyboard shortcut (Alt+z) or Save button located at the bottom of the page.
  8. File will be saved in your local machine (downloads folder by default) as JSON file, use this for editing or reporting purposes.
  9. Select 'Accessibility Conformance Report' to view test result you saved (with at least one valid test result). This is read only page, no edit allowded. To save as printer friendly HTML file, select the keyboard shortcut (Alt+s) or Save button located at the bottom of the page.
  10. You need both test results JSON (for editing) and HTML format(user friendly) for future reference.
